pop 取出并返回数组的最后一个元素： push 在数组末端添加元素： shift 取出数组的第一个元素并返回它： let fruits = ["Apple", "Orange", "Pear"]; alert( fruits.shift() ); // 移除 Apple 然后 alert 显示出来 alert( fruits ); // Orange, Pear unshift 在数组的前端添加元素： let fruits = ["Orange", "Pear"]; fruits.unshift('Apple'); alert( fruits ); // Apple, Orange, Pear 循环 let fruits = ["Apple", "Orange", "Plum"]; // 迭代数组元素 for (let fruit of fruits) { alert( fruit ); } 所以，清空数组最好的方法就是：arr.length = 0;。 toString
